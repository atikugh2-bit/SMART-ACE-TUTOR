# SMART-ACE AI Tutor
### MSRA SJT Preparation Tool by Dr Adam Atiku

## Deploy to Vercel (Recommended — 5 minutes)

1. Go to https://vercel.com and sign up free
2. Click "Add New Project"
3. Upload this folder (or connect GitHub)
4. Framework: Vite
5. Click Deploy
6. Your URL: smart-ace-tutor.vercel.app

## Deploy to Netlify (Alternative)

1. Go to https://netlify.com
2. Drag the entire folder onto the deploy zone
3. Instant URL provided

## Local Development

```
npm install
npm run dev
```
